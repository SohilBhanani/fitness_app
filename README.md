# fitness_app

## Description
An app to calculate the Body Mass Index and Calories Burned.

## Technologies Used
* <b>Flutter</b>
* <b>Firebase authentication</b>
* <b>Firestore</b>

## Usages
* BMI Calculator
* Calories Burned (Revised Harris Benedict Equation)
